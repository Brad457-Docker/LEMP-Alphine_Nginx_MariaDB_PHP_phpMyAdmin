# LEMP stack
a simple LEMP using the *Offical* Docker Repository for Alpine Linux, Nginx, MariaDB, PHP and phpMyAdmin

### for start LEMP: 
``docker-compose up -d``
### for stop LEMP:
``docker-compose stop``
### for remove LEMP:
`` docker-compose down``
